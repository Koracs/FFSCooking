## Available Scripts

In the project directory, you can run:

!!!!!!!!!!!!for backend!!!!!!!!!!!!

### `npm install `
1) open powershell
2) navigate into backend folder
3) run npm install

### `npm start`

1) open powershell
2) navigate into backend folder
3) run npm start

!!!!!!!!!!!!for frontend:!!!!!!!!!!!!

### `npm install `
1) open powershell
2) navigate into frontend folder
3) run npm install

### `npm start`

1) open powershell
2) navigate into frontend folder
3) run npm start
